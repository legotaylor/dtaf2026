# Somnium Reale  

> Somnium Reale contains some minor flashing lights and may not be suitable for some players with photosensitive medical
> conditions.
>
> If you experience lightheadedness, altered vision, twitching, involuntary movements, disorientation, confusion, or temporary loss of awareness, please stop playing immediately and seek medical advice.  

> Mod compatibility is not guaranteed, and I recommend playing this by itself.
>
> Sodium/Iris *will* work, but won't give some of the intended visual effects, and some shaders may break.

## Missing Features

Here is the list of features that didn't make it due to time constraints.  
Some of these could be added in a future update.

- **Blocks/Items**
	- Maple Sign
	- Maple Hanging Sign
	- Stone and Basalt Bricks
	- Entity Observer Block
		- Outputs redstone power when an entity is detected in front of it, power level depends on distance.
- **Shaders**
	- Coloured Lighting
	- Luminance Shaders
		- Luminance would allow for easier customizability of post-effect shaders.
- **Minecraft 26.1**
	- There has been many changes from 1.21.8 → 26.1 which would make updating slightly harder.
	- Once Luminance is updated, I might update the mod and add some of the other missing features.
- **Recipe Book Support**
- **More Entity Variants**
- **Custom Music**
